# Day 17: API Testing & Quality Assurance

This is a minimal Express API project built to demonstrate testing with **Jest** and **Supertest**.
Features:
- Users API (CRUD)
- Posts API (CRUD)
- Search API (cross-resource)
- Health check
- In-memory datastore for fast tests

Install:
```bash
npm install
```

Run server:
```bash
npm start
```

Run tests:
```bash
npm test
```
